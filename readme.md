### Display to Board Mapping

| ST7796S Display Pin | Nucleo-G474RE Pin |
|----------------------|-------------------|
| VCC                 | 3.3V             |
| GND                 | GND              |
| CS                  | PB6              |
| RST                 | PA9              |
| DC/RS               | PC7              |
| MOSI                | PA7              |
| SCK                 | PA5              |
| LED                 | 3.3V             |
| MISO                | PA6              |

### Board to Pins Mapping

| Nucleo-G474RE Pin | Connection         |
|-------------------|--------------------|
| PA0               | ADC Input         |
| PA1               | ADC GND           |
| PA4               | Up Button         |
| PB0               | Down Button       |
| PC1               | Set Button        |
| PC0               | Hold Button       |
